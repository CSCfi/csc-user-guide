# Finnish translation for Docs CSC

CSC’s end user guides and tutorials portal Docs CSC is getting an automatic Finnish translation, powered by a large language model. With this requested feature, the goal is to ensure that Finnish-speaking users can access the same information as English-speaking users, with translations kept continuously up to date. The translation will become a part of the existing Docs CSC service component on 27.04.2026. 


## Background

Over the past few years, we’ve received growing feedback from users asking for Docs CSC content in Finnish. Providing documentation in Finnish also supports CSC’s role as a public service organization. 

At the same time, maintaining the same documentation manually in multiple languages is difficult to scale. To support long term documentation upkeep while keeping content consistent across languages, we have chosen to automate the translation into Finnish and continue maintaining the English documentation as the primary source.


## How it works?

The language can be changed from the :octicons-globe-16: icon in the top banner on Docs CSC
- The English version is the primary source.
- Finnish pages are generated automatically based on the English content.
- Translations are updated daily at midnight. 

Because of this, there may be short periods when the English and Finnish pages differ slightly, for example if the English page has been updated very recently.


## Quality and Feedback

Because of the LLM translation, there might be some quality issues with the translation. To account for this, we have an AI disclaimer in place. If you notice issues or want to give feedback, please [contact us](contact.md).
The translation tool will continue to be developed after the initial publication to ensure an even better experience for the Finnish-speaking users.

Some parts of the documentation require additional development, including:
- Navigation links
- Images and
- Code snippets among text
