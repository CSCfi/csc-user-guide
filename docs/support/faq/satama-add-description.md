# Add Description to repositories

After pushing an image, the project administrator can add information to describe the repository. Adding a description to a repository helps users and teams understand the purpose, contents, and usage of the images stored within it.
## Steps

* Go into the repository and select the **Info** ta
* Click the **Edit** button
* Enter the description. For example, its base image, its intended use, or any relevant deployment notes.
* Click **Save** to save the description.
