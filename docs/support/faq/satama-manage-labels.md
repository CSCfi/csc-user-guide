# Managing Labels

Labels can be used to categorize and organize repositories an images. it can help in quickly understand the purpose or ownership. You can also filter images by labels. Satama has two types of labels:

## 1. Global Level Labels

 System administrators can create, update and delete the global level labels. These can be used by any image under any project. 
    
## 2. Project Level Labels

 Project administrators and system administrators can list, create, update and delete the global level labels. These are mainly managed by project administrators only because of the scope of these labels.
