## JAX 0.4.30 saatavilla LUMIssa, 23.10.2024 { #jax-0.4.30-available-on-lumi,-23.10.2024 }

[JAX](../../apps/jax.md) 0.4.30 on nyt asennettu LUMIin ja asetettu oletusversioksi, kun se aiemmin tehtiin saataville Puhtiin ja Mahtiin. Asennus LUMIin hyödyntää nyt uusia AMD-ajureita ja ROCm 6.0.3:a, jotka asennettiin LUMIn järjestelmäpäivityksen yhteydessä syyskuussa.

`transformers`-paketti ei enää sisälly, koska se tukee virallisesti JAXia vain versioon 0.4.13 asti.

Tästä versiosta alkaen JAXista on "small"- ja "full"-makuja: "full" sisältää JAX ML -ekosysteemin paketit kuten ennen, kun taas "small" sisältää vain JAXin GPU-tuella sekä Tensorflow’n ja PyTorchin vain-CPU-versiot niiden datanlataus- ja -käsittelytyökaluja varten ilman lisäpaketteja. Voit käyttää jälkimmäistä, jos haluat puhtaamman pohjan lisäpakettien asennukseen.

[julkaisutiedot](https://github.com/cp2k/cp2k/releases/tag/v2024.1) uusista ominaisuuksista, korjauksista ja rikkomista muutoksista. Huomionarvoisena muutoksena laskut nyt keskeytyvät varoituksen sijasta, jos SCF-sykli ei konvergoidu. Lisää `IGNORE_CONVERGENCE_FAILURE` -avainsana `&SCF`-osioon ottaaksesi käyttöön vanhan toiminnan.

## NAMD 3.0b6 saatavilla LUMIssa, 12.3.2024 { #namd-3.0b6-available-on-lumi,-12.3.2024 }

Viimeisin AMD-GPU-tuettu NAMD-versio on nyt esiasennettuna LUMIssa. Katso [NAMD-sivultamme](../../apps/namd.md) esimerkit eräajon skripteistä ja suorituskykyhuomiot sekä
[NAMD-verkkosivuilta](https://www.ks.uiuc.edu/Research/namd/3.0/features.html) lista uusista ominaisuuksista, jotka parantavat yhden solmun monen GPU:n simulointien suorituskykyä.

## JAX 0.4.23, 6.3.2024 { #jax-0.4.23,-6.3.2024 }

[JAX](../../apps/jax.md) 0.4.23 on lisätty Puhtiin, Mahtiin ja LUMIin.
JAX-moduulit sisältävät nyt myös vain-CPU-asennuksen PyTorchista, jota toisinaan käytetään JAXin kanssa sen datanlatausominaisuuksien vuoksi, sekä asiaankuuluvat päivitykset kaikkiin aiemmissa JAX-moduuleissa mukana olleisiin Python-paketteihin.

## PyTorch 2.2.1, 1.3.2024 { #pytorch-2.2.1,-1.3.2024 }

[PyTorch](../../apps/pytorch.md) 2.2.1 on lisätty Puhtiin, Mahtiin ja LUMIin. LUMIn moduuli sisältää ROCm-versiot
[FlashAttention-2](https://github.com/ROCm/flash-attention) ja
[bitsandbytes](https://github.com/ROCm/bitsandbytes) -paketeista, sillä niiden lisääminen itse on käyttäjille hankalaa.
[xFormers](https://github.com/facebookresearch/xformers) on lisätty kaikkiin kolmeen järjestelmään aiemmin tuettujen pakettien (esim. DeepSpeed ja Transformers) uusien versioiden lisäksi.
