# ArcGIS { #arcgis }

[ArcGIS software] kattaa laajan valikoiman kehittyneitä GIS-
(Geographic Information System) työkaluja. Sitä voidaan käyttää työpöytä-, palvelin-, verkko-
ja mobiiliympäristöissä.

CSC järjestää ArcGIS-ohjelmistokonsortion Suomen yliopistoille ja
ammattikorkeakouluille. Nykyisen konsortion jäsenet on
lueteltu tämän sivun lopussa. 

!!! note
    CSC:n ArcGIS-ohjelmistokonsortio päättyy 31.12.2025. 

Konsortion jäsenille ovat saatavilla seuraavat tuotteet (tarkemmat tiedot lisenssisopimuksen liitteessä A):

-   ArcGIS Online
-   ArcGIS Pro ja Desktop
-   ArcGIS Enterprise 
-   ArcGIS Drone2map
-   Esri CityEngine
-   ArcGIS Monitor
-   ArcGIS Hub Premium
-   ArcGIS Developer Subscription
-   ArcGIS Premium Apps
-   Virtual Campus -kurssit
-   [Esri Finlandin Suomen tie- ja katuverkkoaineisto](https://wiki.eduuni.fi/display/cscjemma/Accessible+for+ArcGIS+consortium+members). Tarkoitettu
    reititykseen ja geokoodaukseen. Aineisto pohjautuu Digiroadiin, mutta sisältää useita korjauksia ja
    parannuksia. 

Useimmat ohjelmistotuotteet ovat saatavilla *rajoittamattomalle* käyttäjämäärälle. ArcGIS:ää voidaan käyttää opetuksessa, 
tutkimuksessa tai rajallisesti hallinnolliseen työhön. Kaupallinen käyttö on kielletty. 
Ohjelmiston saa asentaa konsortioon kuuluvan korkeakoulun tietokoneille sekä henkilökunnan 
ja opiskelijoiden koneille, ja sitä saa käyttää kotona tai maastotyössä käyttöehtojen mukaisesti. Tarkemmat ehdot
löytyvät dokumentista [ArcGIS site license -sopimus](../img/ArcGIS_education_Institution_terms_and_conditions_2020-2022.pdf).

1.1.2025 alkaen seuraavat ArcGIS Desktop -laajennukset eivät ole saatavilla: Data Interoperability, Aviation Airports, Aviation Charting, Bathymetry, Data Reviewer, Defense Mapping, LocateXT, Maritime,
Production Mapping, Tracking Analyst, Workflow Manager, Location Referencing Desktop (Pipeline) ja Location Referencing Desktop (Roads&Hwy).

    ArcGIS Desktopin kehitys on päättynyt, ja se poistuu virallisesti käytöstä 1.3.2026; lisenssejä ei todennäköisesti ole saatavilla 1.1.2026 alkaen. ArcGIS Desktop -käyttäjiä kehotetaan [siirtymään ArcGIS Prohon](https://support.esri.com/en/arcmap-esri-plan) vuoden 2025 aikana. 

### Asennus { #installation }
-   [Asennusohjeet ArcGIS Prolle](https://pro.arcgis.com/en/pro-app/latest/get-started/install-and-sign-in-to-arcgis-pro.htm) ja [ArcGIS Desktopille](https://desktop.arcgis.com/en/arcmap/latest/get-started/installation-guide/introduction.htm).
-   Asennusta varten tarvitset **asennuspaketin**; kysy sitä oman yliopistosi ArcGIS-yhteyshenkilöltä. ArcGIS Pron asennuspaketin voi myös [ladata ArcGIS Onlinesta](https://pro.arcgis.com/en/pro-app/latest/get-started/download-arcgis-pro.htm#ESRI_SECTION1_F19E35C5A28F44F69A0EA3F464A0E015). 
-   Asenna myös tarvittavat [korjauspäivitykset ja Service Packit](https://support.esri.com/en/Products/Desktop/arcgis-desktop/arcmap/).

### Lisenssi { #license }
ArcGIS-tuotteet on valtuutettava. Lisensointivaihtoehdot konsortion jäsenille:

-   **Nimetyt käyttäjät** (Named users), ensisijainen vaihtoehto ArcGIS Prolle; ei saatavilla ArcGIS Desktopille. Käyttäjällä tulee olla kotiyliopiston ArcGIS Online -tili ja henkilökohtainen ArcGIS Pro -lisenssi. Pyydä tämä oman yliopistosi ArcGIS Online -ylläpitäjältä.  
-   **Samanaikaiskäyttölisenssit** CSC:n lisenssipalvelimelta, ensisijainen vaihtoehto ArcGIS Desktopille. ArcGIS Pron samanaikaiskäyttölisenssit ovat saatavilla vain 31.3.2025 asti. 
Pyydä oman yliopistosi ArcGIS-yhteyshenkilöltä lisenssipalvelimen osoite ja avattavat portit. 
Ohje: [ArcGIS Pro](http://pro.arcgis.com/en/pro-app/get-started/start-arcgis-pro-with-a-concurrent-use-license.htm) 
ja [ArcGIS Desktop](https://desktop.arcgis.com/en/arcmap/latest/get-started/installation-guide/authorization-wizard.htm).
-   **Yksittäiskäyttölisenssi** (Single use), ArcGIS Prolle ja ArcGIS Desktopille. Ei saatavilla vuonna 2025. 

  
!!! note
    CSC:n lisenssipalvelin on käytettävissä vain FUNET-verkosta (Suomen
    yliopistot). Kun työskentelet FUNET-verkon ulkopuolella, esimerkiksi kotona, muodosta ensin yhteys
    omaan yliopistoverkkoosi (VPN). 

#### Samanaikaislisenssien lainaaminen { #borrowing-concurrent-licenses }

ArcGISin Educational site -lisenssi sisältää myös mahdollisuuden lainata
lisenssejä 20 päiväksi. Lainatulla lisenssillä voit työskennellä
ilman internetyhteyttä tai FUNET-verkon ulkopuolella.

Lisenssi lainataan sovelluksella [ArcGIS Administrator](http://desktop.arcgis.com/en/arcmap/latest/get-started/administer/borrowing-and-returning-licenses.htm) tai [ArcGIS Prossa](https://pro.arcgis.com/en/pro-app/get-started/start-arcgis-pro-with-a-concurrent-use-license.htm#ESRI_SECTION1_9E03C2B7A83C4FE4A41408EE7E0498CB):

-   Lainaa aina perustuote, eli ArcGIS Desktop tai ArcGIS Pro.
-   Kaikki laajennukset on lainattava erikseen; voit valita tarvitsemiesi laajennusten joukosta. Varmista erityisesti ArcGIS Prossa, että lainaat kaikki tarvitsemasi laajennukset. Laajennuksia ei voi lisätä jo lainattuun perustuotteeseen.
-   Vuoden lopussa lainatut lisenssit palautuvat aina 31.12., koska lisenssi uusitaan vuosittain.
-   Lainatun lisenssin palautus ei toimi ArcGIS Prossa. Lisenssi palautuu automaattisesti laina-ajan viimeisenä päivänä.

## ArcGIS Online { #arcgis-online }

[ArcGIS Online] on alusta karttojen, sovellusten ja paikkatiedon tuottamiseen ja jakamiseen. ArcGIS Online on käytettävissä konsortion jäsenille – kukin korkeakoulu voi luoda enintään 5000 käyttäjää ja käyttää vuodessa 50 000 [ArcGIS
Online -krediittiä](http://www.esri.com/software/arcgis/arcgisonline/credits). Liittymisohjeet saat oman yliopistosi ArcGIS-yhteyshenkilöltä.

Ohjeita ArcGIS Onlinen ylläpitäjille:

- [Kutsu ja lisää käyttäjiä](http://doc.arcgis.com/en/arcgis-online/administer/invite-users.htm)
- [ArcGIS-lisenssien myöntäminen ArcGIS Onlinessa](http://pro.arcgis.com/en/pro-app/get-started/manage-licenses.htm)

## Virtual Campus { #virtual-campus }

Virtual Campus sisältää laajan valikoiman GIS-aiheisia verkkokursseja,
painopisteenä ESRI:n työkalut ja teknologiat. Monet Virtual Campus -
kurssit ovat maksuttomia, ja lisäksi konsortion jäsenillä on pääsy kaikkiin
Web Course- ja Training seminar -tyyppisiin kursseihin.

-   [Lista kaikista Virtual Campus -kursseista](https://www.esri.com/training/catalog/search/)

Osallistumiseen tarvitset ESRI Global Account -tilin. Maksullisille kursseille tarvitset lisäksi liittämisen konsortion ArcGIS-tiliin; pyydä sitä
CSC:ltä.

## Suoritusvinkkejä ArcGIS-työpöytäsovelluksille { #performance-tips-for-arcgis-desktop-software }

* Sekä ArcMapissa että ArcGIS Prossa on useita toimintoja, jotka voivat hyödyntää rinnakkaislaskentaa, ja erityisesti ArcGIS Prossa tilanne paranee jatkuvasti. Tarkista siis verkkodokumentaatiosta ajantasainen tilanne kunkin työkalun osalta. Ota rinnakkaiskäsittely käyttöön asettamalla [Parallel processing factor](https://pro.arcgis.com/en/pro-app/tool-reference/environment-settings/parallel-processing-factor.htm) oikein ympäristöasetuksissa tai käytetyn työkalun asetuksissa.
* Parhaan suorituskyvyn saavuttamiseksi ArcMapissa ota käyttöön [64-bit-background Geoprocessing](http://desktop.arcgis.com/en/arcmap/latest/analyze/executing-tools/64bit-background.htm). 
* Työtilan `memory / in_memory` käyttäminen geometriakohteiden sijaan on nopeampaa, mutta jos tarvitset muistia myös laskentaan ja sinulla on suuri aineisto ladattuna, muistia ei välttämättä ole riittävästi. Ohje: [ArcGIS Pro](https://pro.arcgis.com/en/pro-app/latest/help/analysis/geoprocessing/basics/the-in-memory-workspace.htm) / [ArcGIS Desktop](http://desktop.arcgis.com/en/arcmap/latest/analyze/modelbuilder/the-in-memory-workspace.htm).
* Hyödynnä ArcPy-skriptejä, joissa rinnakkaistaminen tehdään Pythonilla, esimerkiksi Multiprocessing- tai Parallel-kirjastoilla.
   - [Arcpy-multiprocessing-esimerkkejä](https://www.e-education.psu.edu/geog489/node/2261)
   - [Parallel Python: Multiprocessing With ArcPy](https://www.youtube.com/watch?v=KAzCG6C8-7g), [esimerkkejä](https://github.com/nRajasekar92/DevSummit-2017)
   - [Blogi: Using Arcpy with multiprocessing](https://pythongisandstuff.wordpress.com/2013/07/31/using-arcpy-with-multiprocessing-part-3/)
   - [Esimerkki: Producing Building Shadows](http://gis.stackexchange.com/questions/19935/port-producing-building-shadows-avenue-code-to-arcgis-10/20352#20352)

## ArcGIS CSC:n laskentaympäristössä { #arcgis-in-cscs-computing-environment }

*   ArcGIS-työpöytätyökalut ArcGIS Pro ja ArcMap ovat saatavilla vain Windows-käyttöjärjestelmälle, mikä tekee niiden asennuksesta mahdotonta CSC:n superkoneisiin ja hyvin haastavaa cPouta-pilveen, jossa Windows-tuki on rajallinen.
*   ESRI tarjoaa [laajan valikoiman palvelintuotteita](https://enterprise.arcgis.com/en/server/latest/get-started/windows/about-arcgis-server-licensing-roles.htm) suurimittakaavaiseen paikkatietoanalyytiin. Nämä ArcGIS Server -tuotteet tukevat yleensä Linuxia ja voidaan asentaa cPoutaan.
*   [ArcPy-skriptejä](https://pro.arcgis.com/en/pro-app/latest/arcpy/get-started/what-is-arcpy-.htm) varten CSC:n ympäristössä paras vaihtoehto on asentaa ArcGIS Server ja ArcPy cPoutaan. ArcGIS Serverin ArcPy voi poiketa hieman ArcGIS-työpöydän ArcPysta. Asennusohjeet löytyvät [GitHubista](https://github.com/csc-training/geocomputing/tree/master/pouta/arcpy).
*   [ArcGIS Python API](https://developers.arcgis.com/python/) on saatavilla Puhtissa moduulina `arcgis-python-api`. Moduuli perustuu [ESRI:n arcgis_learn -metapakettiin](https://developers.arcgis.com/python/latest/guide/install-and-set-up/deep-learning/) ja sisältää myös syväoppimisen kirjastoja. Katso ideoita ja esimerkkejä: [ESRI:n ArcGIS Python API -näytteet](https://developers.arcgis.com/python/samples/). Osa ArcGIS Python API:n toiminnoista edellyttää kirjautumista oman organisaatiosi ArcGIS Onlineen. ArcGIS Python API asennettiin Puhtiin hyödyntämällä [Tykky-työkalun conda-containerize-toiminnallisuutta](../computing/containers/tykky.md) käyttäen tätä [ArcGIS Python API -conda-ympäristötiedostoa](https://github.com/csc-training/geocomputing/blob/master/supercomputer_installations/arcgis-python-api-2.1.0.yml).
