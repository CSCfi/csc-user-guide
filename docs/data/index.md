# Data

CSC provides services and instructions for managing research data throughout
the data lifecycle: planning data management, finding datasets, storing data,
moving data between services, and publishing data.

CSC also provides support for sensitive data workflows.
See the [sensitive data section](./sensitive-data/index.md) in the docs.

This section helps you choose the right instructions for your data-related
task.

## [Working with data](datasets/dataset-sources.md)

Start here if you are planning how to manage, document, store, find or publish
research data.

## [Moving data](moving/index.md)

Use these instructions when you need to transfer data between your local
workstation, CSC supercomputers, Allas, IDA or other CSC services.

## [Allas object storage](Allas/index.md)

Allas is CSC's object storage service for storing and sharing research data.
Use the Allas instructions when you need persistent object storage, or need to
host and share datasets between projects, or between multiple CSC platforms (e.g. cPouta and Roihu).
