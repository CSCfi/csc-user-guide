# SD Desktop: major service upgrade (CSC project- Academic type)

## Overview

The SD Desktop service has been upgraded to a new version and the changes and new features are not fully backward comaptible. This guide explains the actions you need to take to continue accessing your virtual desktop and to restore import/export functionality using the new Data Gateway application. 

Your next steps depend on whether your virtual desktop currently has a volume attached.

## Step 2: choose the correct option for your needs

| Scenario | Recommended Action | Advantages | Disadvantages | Instructions |
|----------|------------------|------------|---------------|--------------|
| Virtual desktop **has a volume** and you want minimal effort | Stay on the current virtual desktop and import the new data gateway | - No migration required<br>- Keep using your existing environment without reinstalling the software<br>- Quick and simple solution | - No access to new features<br>- Cannot attach or detach volumes while the virtual desktop is running<br>- Limited future compatibility<br>- Cannot attach multiple volumes to the same virtual desktop<br>- All project members must import the new data gateway | [Step by step guide](./sd-desktop-major-upgrade-option1.md) |
| Virtual desktop **has a volume** and you plan to continue the analysis long-term | Move the volume to a new virtual desktop | - Access to all new features<br>- Data gateway is available and up to date by default<br>- Fully security-patched environment<br>- Ability to manage volumes while the desktop is running<br>- Supports attaching multiple volumes<br>- Better long-term solution | - Requires manual work<br>- Requires reinstalling the software<br>- Migration effort and team coordination needed | [Step by step guide](./sd-desktop-major-upgrade-option2.md) |
| Virtual desktop **has no volume** | Move to a new virtual desktop | - Clean and stable environment<br>- Fully security-patched environment<br>- Avoid risk of system issues<br>- Access to the latest features | - Requires exporting all files<br>- Requires setting up a new environment | [Step by step guide](./sd-desktop-major-upgrade-option2.md) |


!!! Note
   If you need help choosing the right option or following the steps in the guides, please contact us. We are happy to provide online support with step-by-step guidance.
