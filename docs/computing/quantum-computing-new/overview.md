# Quantum computing at CSC

CSC provides resources for both quantum computing using real quantum computers and quantum simulators utilising the available HPC capabilities.

- **[Quantum computers](quantum-computers/overview.md)** — run circuits on
  physical quantum processors (QPUs) available through CSC supercomputers
- **[Quantum simulators](quantum-simulators/overview.md)** — simulate quantum
  circuits on classical HPC resources, ideal for developing and testing
  algorithms before running them on real hardware.

New to quantum computing at CSC? Start with the
[quantum computers overview](quantum-computers/overview.md) to see the available
devices and how to get started.
